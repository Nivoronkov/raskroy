# Раскрой из спецификации

Автоматизация раскроя линейного проката: спецификация Компас → расчёт раскроя.

## Что где

```
raskroy/
├── raskroy.py            ← главный скрипт: запуск всего процесса
├── sp_to_cutlist.py      ← конвертер спецификации Компас → формат калькулятора
├── requirements.txt      ← список библиотек (с версиями)
├── README.md             ← этот файл
├── ИНСТРУКЦИЯ.md          ← подробная установка с нуля
│
├── smart_cut_app/        ← калькулятор раскроя (с GitHub Nivoronkov/smart_cut_app)
│   ├── materials_catalog.json   ← справочник склада (ведётся вручную)
│   └── leftovers_db.json        ← справочник остатков
│
└── tests/                ← тесты (защита от регрессий)
    ├── test_razbor.py
    └── data/karkas_etalon.xls   ← эталонная спецификация для теста
```

## Быстрый старт

```cmd
cd C:\raskroy
venv\Scripts\activate
python raskroy.py "спецификация.xls"
```

Результат — два файла рядом со спецификацией:
- `..._вход.xlsx` — что система поняла (для проверки глазами);
- `..._раскрой.xlsx` — готовая карта раскроя.

## Прогон тестов

После любой правки конвертера:
```cmd
venv\Scripts\activate
pytest
```
Зелёно — разбор эталона не сломан. Красно — правка что-то поломала.
